## Setup and Running

1. **Create a virtual environment:**

   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. **Install dependencies:**

   ```bash
   pip install -r requirements.txt
   ```

3. **Run the `fuzzy-set-object-distance.py` file:**
   ```bash
   python fuzzy-set-object-distance.py
   ```
